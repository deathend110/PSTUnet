# **Figure 2：Bi-QAG-PST Network（整体结构图）**
---

## **整体布局（核心：三行结构）**

```
Row 1: Input sequence  x1  x2  x3 ... xT
Row 2: Forward flow   → → → →
Row 3: Backward flow  ← ← ← ←
Row 4: Fusion output  y1  y2 ... yT
```

---

## **① 输入序列**

画一排：

```
[x1] [x2] [x3] ... [xT]
```

```
conf1, conf2, ...
```

---

## **② Forward Branch（上方）**

```
x1 → cell → h1 → cell → h2 → ... → hT
```

画法：

* 每个 timestep 一个小模块（简化版 cell）
* 箭头向右
* 标：

```
Forward Propagation
```

---

## **③ Backward Branch（下方）**

```
xT → cell → hT → cell → hT-1 → ... → h1
```

画法：

* 从右往左
* 标：

```
Backward Propagation
```

---

## **④ 双向对齐（关键点）**

在每个 t：

```
h_f(t)   h_b(t)
    \     /
     Concat
       ↓
     1×1 Conv
       ↓
      y_t
```

画法：

* 上下两条线汇合
* 中间一个融合模块
